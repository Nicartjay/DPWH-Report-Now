DPWH Data Cache Refresh Tool

Fetches project data from the DPWH API per city and creates cache files. Uses the same city list and search logic as the app (from provinces.js).

How it works:
- Uses cities from provinces.js (same data the app uses)
- Appends "+city" to search query if "city" is not in the name (same logic as app)
- Creates one JSON file per city: /data/cache/{province}/{city-search}.json
- Must be run from http://localhost:3000 (DPWH API only works from browser)

After fetching: Download the ZIP and extract to public/data/cache/, then commit and push.

Step 1: Select Provinces

Select which provinces to fetch. These come from provinces.js.

Step 2: Fetch Data

Fetches projects for each city using the same query format as the app.
